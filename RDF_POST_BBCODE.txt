[SIZE=5][B]DIY V1 Gen2 Display - Quick Update[/B][/SIZE]

[B]iOS Note:[/B] Also tested with good results using V1 Companion and V1 Driver apps. Connects to Dukes R8 app but I believe @dukes is still building in V1 functionality.

[B]Install:[/B]
[CODE]
git clone https://github.com/ajmdroid/v1g2_simple
cd v1g2_simple
git checkout add/muted_alert_color

# Build (Waveshare 3.49 AMOLED)
pio run -e waveshare-349

# Upload to device
pio run -e waveshare-349 -t upload
[/CODE]
